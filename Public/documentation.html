API Endpoints

1. Get all movies

  • URL: /movies
  • Method: GET
  • Description: Returns a list of all movies.

2. Get a single movie by title

  • URL: /movies/:title
  • Method: GET
  • Description: Returns data about a single movie by title (description, genre, director, image URL).

3. Get data about a genre by name

  • URL: /genres/:name
  • Method: GET
  • Description: Returns data about a genre (description) by name.

4. Get data about a director by name

  • URL: /directors/:name
  • Method: GET
  • Description: Returns data about a director (bio, birth year, death year) by name.

5. Register a new user

  • URL: /users
  • Method: POST
  • Description: Registers a new user.

6. Update user info (username)

  • URL: /users/:username
  • Method: PUT
  • Description: Updates user info (username).

7. Add a movie to a user’s list of favorites

  • URL: /users/:username/movies/:movieId
  • Method: POST
  • Description: Adds a movie to the user’s list of favorites.

8. Remove a movie from a user’s list of favorites

  • URL: /users/:username/movies/:movieId
  • Method: DELETE
  • Description: Removes a movie from the user’s list of favorites.

9. Deregister a user

  • URL: /users/:username
  • Method: DELETE
  • Description: Deregisters a user.
